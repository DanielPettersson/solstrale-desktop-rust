# solstrale-desktop-rust

A desktop UI for the Solstråle path tracer

## How to build (Fedora 39)

1. Install Rust: https://rustup.rs/
2. Clone this repo and go to root folder
3. Run `./build.sh` and take a cup of coffee
4. Start the program by `target/release/solstrale-desktop`
